# Design Direction

Goals:
- professional
- trustworthy
- clean
- warm but not overly playful
- stronger than current Wix design
- easy to scan
- mobile-friendly

Keep:
- current Daroda branding direction
- emphasis on booking CTA
- planner lead magnet

Improve:
- visual consistency
- typography hierarchy
- spacing
- section structure
- page-level SEO readiness

Avoid:
- overly flashy effects
- startup/SaaS look
- complicated interactions in first scaffold
