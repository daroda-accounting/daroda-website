# Project Overview

Daroda Accounting is migrating from a Wix website to a custom React + Vite website.

Goals:
- preserve current business flow
- improve SEO scalability
- improve maintainability
- keep Calendly as the booking system
- keep the current Wix site live until migration is ready

Initial build priority:
- visible, professional marketing site shell
- clean routing
- reusable components
- homepage + services + about + contact + booking pages
- no CRM/admin features yet
