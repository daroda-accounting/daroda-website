## Weekly Project Discipline System

3 weekly docs / updates--no more.

### 🧾 1. Weekly Status Snapshot (MANDATORY)

Contains:
- What was completed
- What is blocked
- Decisions needed
- Risks noticed
- Plan for next week

### 📊 2. Time Log / Phase Progress Tracker (Internship requirement)

“Time spent by phase.”

Example:

Phase 0 planning → 12 hrs
Asset extraction → 6 hrs
Repo setup → 3 hrs

### ⚠️ 3. Risk / Issue Log Updates (only if needed)

Don’t rewrite the Risk Register weekly.

(Example) You just append:
“New risk discovered: Calendly iframe mobile overflow”

Milestone docs, not weekly:

- Migration Checklist
- Post-launch Validation
- Growth KPI Report
